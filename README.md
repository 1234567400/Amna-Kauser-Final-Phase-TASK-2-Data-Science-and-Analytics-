# Objective
Segment customers based on income and spending behavior using unsupervised learning, and design targeted marketing strategies.

# Approach

# Data Analysis: Explored 200 customers with features like age, gender, income, and spending score.
Feature Selection: Used income and spending score, scaled using StandardScaler.
Clustering: Identified optimal clusters (k=5) using Elbow Method and Silhouette Score.
Model: Applied K-Means to create 5 segments.
Visualization: Used PCA and t-SNE for 2D/3D cluster visualization.
Strategy: Designed marketing strategies for each segment.

# Results and Insights

Optimal Segments: 5 clusters (Silhouette Score: 0.555).
Customer Segments:
Average Consumers: Medium income/spending → promotions & bundles
Luxury Enthusiasts: High income/spending → VIP & exclusive offers
Impulsive Spenders: Low income/high spending → flash sales & BNPL
Cautious Spenders: High income/low spending → personalized & value-based offers
Frugal Customers: Low income/spending → budget deals & discounts
Insight: t-SNE showed clearer cluster separation than PCA, indicating non-linear patterns.
